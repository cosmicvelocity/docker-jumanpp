# jumanpp
jumanpp, juman, knp を含むイメージです。

- [JUMAN](http://nlp.ist.i.kyoto-u.ac.jp/index.php?JUMAN)
- [JUMAN++](http://nlp.ist.i.kyoto-u.ac.jp/index.php?JUMAN++)
- [KNP](http://nlp.ist.i.kyoto-u.ac.jp/index.php?KNP)

辞書も含んでいるのですぐに試すことができますが、イメージサイズはかなり大きくなっており、 pull した状態で 5.19GB 前後になります。
